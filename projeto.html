Aprendendo a Formatar texto

Negrito ou Strong

Para usar negrito você pode usar o <b> de Bold ou o Strong,
mas o mais recomendado é o strong.

Itálico ou Emphasis

Para usar o itálico, você pode usar o <i> de Itálico ou o Emphasis,
mas o mais recomendado é o Emphasis.

Underline

Para usar o Underline, você utiliza o <u> de Underline.

Taxado

Para taxar um texto, você utiliza o <strike>, de Taxado.

Listas Ordenadas e Não Ordenadas

Para criar uma lista não ordenada basta usar <ul>, e dentro da tag usar "< li >".
li são as tags de list item.

Exemplo de Lista Não Ordenada:
  • Home
  • Quem Somos?
  • Contatos
  • Projetos

Para criar uma lista ordenada basta usar <ol>, e dentro da tag usar "< li >".
li são as tags de list item.

Exemplo de Lista Ordenada:
  1. Melhor cadeira do primeiro semestre?
  2. Gênero de Jogo favorito
  3. Melhor IA para gerar imagem

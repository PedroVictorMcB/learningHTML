Página Principal

Aqui estou aprendendo a gerar links e criar uma página principal.
Usei a tag <a>, para gerar um link. Dentro dela vem o href, utilizado para referenciar
o diretório destino. Como estou usando pastas dentro do próprio projeto, não precisei passar o endereço do
diretório em href, nos links. Mas caso estivesse usando uma pasta localizada em outro diretório,
então precisaria passar o caminho do diretório em href.
